# Console Menu Application (C)

A terminal-based menu system with keyboard navigation and screen management.

## Features

- **Interactive menu** - Navigate with arrow keys
- **Screen management** - Multiple screens (menu, content display)
- **Input handling** - Keyboard input processing
- **Terminal UI** - Uses custom terminal library for display
- **Modular design** - Separated into app, menu, screen, input, and terminal modules

## Components

- `app` - Main application logic and flow
- `menu` - Menu rendering and selection handling
- `screen` - Screen state management
- `input` - Keyboard input processing
- `terminal` - Low-level terminal control

## Usage

Run the application and use arrow keys to navigate the menu, Enter to select options.

## Architecture

Modular C application demonstrating separation of concerns with header/implementation file pairs for each component.
